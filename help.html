Pencil:

Select pencil tool to draw lines. click and drag to draw along the grid lines.

Eraser:

Select eraser tool to erase lines. click and drag to erase along the grid lines.

Text:

Select text tool to add text. click and drag text to move. Double click text to delete.

Components:

Select components. Click on a grid square to add component. Double click component to delete.

Note: While the text tool is selected. double clicking components will not delete them.

Clear:

Click "clear" button to clear work area.

Download:

Click "download" button to download PNG image.
